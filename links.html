Links of Interest

Computing

  • 0x0.st: cURL file sharing service.
  • 9front: A fork of Plan 9 From Bell Labs that is well maintained.
  • Ataraxia GNU/Linux: A Linux-based OS designed with privacy, security, and simplicity in mind.
  • Brainfuck Algorithms: A list of algorithms implemented in brainfuck.
  • brainfuck.org: Various brainfuck-related items.
  • cat-v: A mysterious group that maintains a wonderful collection of papers and manpages related to UNIX and Plan 9, among other things.
  • CHIP-8 Archive: A collection of ROMs for the CHIP-8 and many of its derivatives.
  • CHIP-8 Reference: A detailed reference for pretty much all things CHIP-8.
  • coreboot: FOSS boot firmware.
  • The Demoscene: Webportal for all things "demos".
  • Esolang: The esoteric programming languages wiki.
  • glaucus: An independent source-based Linux distribution based on musl libc and toybox.
  • h-node: A hardware database for linux-libre (Linux kernel with no proprietary blobs).
  • LandChad: Various tutorials about building an independent online presence.
  • libreCMC: Linux distro for embedded computers (such as routers).
  • LineageOS: An open-source Android distribution.
  • Plan 9 From Bell Labs: Plan 9, UNIX's successor.
  • RFC Index: Documentation for most internet standards.
  • Parabola GNU/Linux-Libre: Arch Linux without any proprietary packages or kernel drivers.
  • pimutils: A small collection of CLI utilities for managing calendars, tasks, and synchronizing data.
  • SDF.org: A public access UNIX system. Anyone can make a free account on this server.
  • SMAP: A document describing a simpler alternative to IMAP.
  • OSDev Wiki: Extensive information about creating operating systems.
  • TLDRLegal: Provides readable summaries of software licenses.
  • wiby: A search engine for the classic web.
  • XDG Base Directory Support List: A list of software that supports the FreeDesktop XDG Base Directory Specification and how to modify others in order to meet the specification.

Other

  • The 88x31 GIF Collection: A large collection of 1990s/2000s style web buttons.
  • Librivox: A collection of public domain audiobooks.
  • WebSDR Index: A list of software-defined radio hosts.
  • YouTube Film Playlist: A collection of interesting film scenes and video essays about film.
  • YouTube History Playlist: Some notable YouTube videos regarding all aspects of history.
